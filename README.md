# 项目介绍

这是对 Golang 的 Gin 框架的一个简单封装，用于开发后端api，旨在创建一个开箱即用的基础框架。

技术栈为 Golang + Gin + MongoDB。

## 项目功能

本项目将会对以下功能进行封装：

- [ ] 全局配置（如日志等级、后端地址等）
- [ ] 日志系统
- [ ] MongoDB 数据库访问
- [ ] 登录接口和 jwt 鉴权中间件
- ...
